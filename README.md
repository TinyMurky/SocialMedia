## 基本介紹
此網站為實做社群網站中My Friend頁面之相關功能，可以點擊使用者頭像觀看個人資訊，並新增至My Friend清單。

## 實做功能
- 使用者頭像卡片可拖移更換位置
- 每頁10位使用者，可使用下方頁籤換頁
- 右上角有搜尋功能，可於輸入時即時搜尋，也可按下搜尋建搜尋
- 點擊頭像卡片可以更進一步了解使用者資訊
- 點擊愛心可將使用者加到My Friend清單
- My Friend清單中若點擊愛心可取消朋友關係
- 主頁之搜尋、換頁與點擊頭像功能於My Friend中也可使用
